**目标**：从"基础RAG"升级为"进阶RAG"，展示数据处理与检索优化能力

#### Week 1：检索策略优化

**Day 1-2：混合检索**

**Day 3-4：Rerank 重排序**

## 📊 技术选型

### RAG 系统升级

| 功能 | 技术方案 | 备选方案 |
|------|---------|---------|
| 混合检索 | rank_bm25 + Chroma | Elasticsearch |
| Rerank | BGE-Reranker | Cohere Rerank |
| Query Rewrite | LLM 提示词 | 专门训练的模型 |
| HyDE | LLM 生成假设文档 | - |
| 评测 | RAGAS | TruLens |

### Agent 项目

| 功能 | 技术方案 | 备选方案 |
|------|---------|---------|
| Agent 框架 | LangChain | LlamaIndex |
| 代码执行 | subprocess + 沙盒 | Docker 容器 |
| 搜索工具 | Tavily API | DuckDuckGo |
| 前端 | HTML + SSE | React |
| 部署 | Docker + 腾讯云 | Vercel |
