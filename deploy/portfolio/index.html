我的项目

AI 智能问答助手

基于腾讯云 CloudBase 的 Serverless AI 问答应用。支持自然语言对话，自动保存历史记录。采用云函数 + 云数据库架构，无需管理服务器。

Node.js CloudBase DeepSeek Serverless
🔗 在线体验 📖 技术解析
📚

RAG 文档问答系统

基于 RAG 架构的文档问答系统。支持用户上传文档，基于文档内容进行语义问答。使用 LangChain + Chroma + DeepSeek 实现向量检索和智能问答。

Python FastAPI LangChain Chroma DeepSeek
📁 源代码 📖 技术解析
🔬

RAG 进阶优化版

在现有 RAG 系统基础上进行深度优化。实现混合检索（向量 + BM25）、Rerank 重排序、Query Rewrite 查询重写，并提供评测报告。

rank_bm25 BGE-Reranker RAGAS HyDE
📋 升级规划
🤖

数据分析 Agent

基于 Function Calling 的数据分析 Agent。支持用户上传 CSV/Excel 数据，Agent 自动理解数据结构，编写代码进行分析和可视化，生成分析报告。

LangChain Function Calling Pandas Echarts
📋 升级规划
